## The Stack

### Frontend

- **React 19** + **React Compiler**
- **TanStack Start**
- **TanStack Form**
- **Tailwind CSS 4** + **BaseUI** + **shadcn/ui**
- **Zod v4**
- **Lexical**

### Backend & Auth

- **Better Auth**
- **Drizzle ORM**
- **PostgreSQL**
- **Server Functions**

### DX & Infrastructure

- **Vite+**
- **Oxc & Oxfmt**
- **T3Env**

## Getting Started

```bash
pnpm install
pnpm dev
```
